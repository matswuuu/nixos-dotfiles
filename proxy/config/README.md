# Local proxy configs

Put your private AmneziaWG/WireGuard configs here:

- `de.conf`
- `jp.conf`
- `se.conf`
- `fi.conf`
- `pl.conf`

These files contain private keys, endpoints, and addresses, so `*.conf` files in
this directory are ignored by Git. Keep only sanitized `*.example` files in the
repository.
